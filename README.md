# Task Management System 🚀

A modern full-stack Task Management System developed using the MERN stack that helps organizations and teams efficiently manage projects, assign tasks, monitor progress, and streamline workflow.

## ✨ Features

- User Authentication with JWT
- Role-Based Access Control (Admin / User)
- Create, Update, Delete Tasks
- Project Management Module
- Task Assignment to Team Members
- Task Progress Tracking
- Secure MongoDB Atlas Cloud Database
- Responsive React Frontend
- RESTful API Integration

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS / Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JSON Web Token (JWT)

## 🌐 Deployment
- Backend deployed on Railway
- Frontend deployed on Railway/Vercel

## 📌 Purpose
This project was built to demonstrate a complete industry-level MERN stack application with authentication, database integration, cloud deployment, and real-time task management functionalities.
